- "태그 ol":  list-style-type: lower-alpha; (a, b, c...)
 
+ block <-> inline
  - block: 1줄을 모두 사용하는 구조. (h, ul, ol, li, table, tr, div...)
  - inline: 1줄에 다음요소와 이어지는 구조 (th, td, img, span)
  - inline-block요소: 1줄에 다음요소와 이어지는 구조(크기지정가능)
